Tích hợp

Cài đặt OpenCode

Cấu hình OpenCode Agent để hoạt động liền mạch với cổng API hợp nhất của Simple Router.

1

Giới thiệu

OpenCode là một agent AI nguồn mở mạnh mẽ chạy trong terminal của bạn. Bằng cách kết nối nó với Simple Router, bạn có thể điều phối nhiều model thông qua một cấu hình duy nhất.

Lợi ích

  • Mã nguồn mở & Ưu tiên cục bộ
  • Cấu hình Provider linh hoạt
  • Truy cập toàn bộ hệ thống tệp
2

Cài đặt

Cài đặt OpenCode CLI toàn cục.

npm install -g opencode-ai
pnpm add -g opencode-ai
yarn global add opencode-ai
3

Cấu hình

Tạo tệp cấu hình để trỏ OpenCode tới Simple Router của bạn.

~/.config/opencode/opencode.json
~/.config/opencode/opencode.json
%USERPROFILE%\.config\opencode\opencode.json
opencode.json content 
{
  "defaultProvider": "simple-router",
  "providers": {
    "simple-router": {
      "type": "openai",
      "baseUrl": "http://simpleverse.io.vn/router/v1",
      "apiKey": "any-key-works",
      "model": "claude-3-5-sonnet"
    }
  },
  "theme": "dark"
}
