Tích hợp Claude Code

Hướng dẫn toàn diện để thiết lập và tối ưu hóa Claude Code CLI của Anthropic với Simple Router.

1

Giới thiệu

Claude Code là một công cụ CLI mang tính cách mạng cho phép Claude tương tác trực tiếp với codebase của bạn, thực thi các lệnh shell và giải quyết các vấn đề phức tạp ngay trong terminal.

Tính năng

  • Đọc/Ghi ngữ cảnh code
  • Chạy lệnh shell
  • Tìm kiếm toàn bộ project
2

Cài đặt Nhanh

Cài đặt Claude Code toàn cục thông qua trình quản lý gói ưa thích của bạn.

Global
npm install -g @anthropic-ai/claude-code
pnpm add -g @anthropic-ai/claude-code
yarn global add @anthropic-ai/claude-code
3

Cấu hình

Để sử dụng Claude Code với định tuyến tùy chỉnh, bạn cần tạo hoặc cập nhật tệp cài đặt toàn cục. Vị trí tệp thay đổi theo hệ điều hành:

~/.claude/settings.json
~/.claude/settings.json
%USERPROFILE%\.claude\settings.json
settings.json content 
{
  "env": {
    "ANTHROPIC_AUTH_TOKEN": "your-secret-password",
    "ANTHROPIC_BASE_URL": "http://simpleverse.io.vn/router",
    "ANTHROPIC_DEFAULT_HAIKU_MODEL": "gemini-3-flash-preview",
    "ANTHROPIC_DEFAULT_OPUS_MODEL": "gemini-claude-opus-4-5-thinking",
    "ANTHROPIC_DEFAULT_SONNET_MODEL": "claude-sonnet-4-5",
    "CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC": "1"
  },
  "alwaysThinkingEnabled": true
}

Gợi ý: Bạn cũng có thể thiết lập các biến môi trường này trực tiếp trong cấu hình shell của mình (như .zshrc hoặc .bashrc) để linh hoạt hơn.
