# Simple Router

Cổng API hợp nhất cho các AI coding agent. Một endpoint duy nhất để truy cập Claude, Gemini, GPT và nhiều model khác.

## Tính năng

- **Truy cập hợp nhất** - Một API cho tất cả model AI
- **OpenAI-compatible** - Tương thích với mọi công cụ hỗ trợ OpenAI API
- **Hiệu năng cao** - Tối ưu độ trễ và caching thông minh

## Model hỗ trợ

- Claude Opus 4.6, Claude Sonnet 4.5
- Gemini 3 Pro, Gemini 3 Flash
- GPT-OSS 120B
- Và nhiều model khác

## Tích hợp

Claude Code • Cursor • OpenCode • Kilo Code • Roo Code • Cline
