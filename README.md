# Simple Router

Cổng API hợp nhất cho AI Coding Agent và OpenClaw. Một endpoint duy nhất để truy cập GPT, Qwen và nhiều model khác.

## Tính năng

- **Truy cập hợp nhất** - Một API cho tất cả model AI
- **OpenAI-compatible** - Tương thích với mọi công cụ hỗ trợ OpenAI API
- **Hiệu năng cao** - Tối ưu độ trễ và caching thông minh

## Model hỗ trợ

- GPT 5.4, GPT 5.3 Codex, GPT 5.2
- Qwen 3.5 Plus, Qwen3 Coder, Qwen3 Vision
- Và nhiều model khác

## Tích hợp

Claude Code • OpenCode • Kilo Code • Roo Code • Cline • OpenClaw
